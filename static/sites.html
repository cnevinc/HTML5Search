Sites

A collection of the sites indexed and aggregated for the best HTML5 search engine on the web. (hint. It is this site)

In the sake of openness, you have the ability to add a site to this collection, fork the project on and edit the http://github.com/PaulKinlan/HTML5Search/tree/master/static/sites.html file. Once you have edited the file, create a pull request and I will review it - all changes and decisions will thus be public and audit-able.
